# Model Experiments

## Model Used
Random Forest Classifier

## Features Considered
- URL Length
- Dot Count
- Hyphen Count

## Why Random Forest?

Random Forest was selected because it is easy to implement, performs well on classification problems, and helps reduce overfitting by combining multiple decision trees.
